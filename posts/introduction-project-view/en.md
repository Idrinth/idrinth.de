# Project View - an introduction

I was busy looking for a decent way of sharing my public but hobby project progress with the world and was unable to find something that fullfilled all my requirements properly.

It needed to provide public view on a Kanban board, a Release history and Time Tracking, while at the same time not being insanly bloated or expensive. Seemingly a near impossible set of requirements.

Another thing that broke the usual boards was the multi-project overview required, categories and sub-categories necessary to filter stuff reasonably.

I ended up asking Claude for an implementation and have been asking it on and off through the day until it ended up with something that is decent enough for now. I'm thinking of properly cleaning it up, but am not sure if there is a need or desire for yet another project management tool.

If there is, feel free to mention it to me!
